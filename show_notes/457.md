**Michael #1: [httptap](https://httptap.dev)**

- Rich-powered CLI that breaks each HTTP request into DNS, connect, TLS, wait, and transfer phases with waterfall timelines, compact summaries, or metrics-only output.
- Features
  - **Phase-by-phase timing** ‚Äì precise measurements built from httpcore trace hooks (with sane fallbacks when metal-level data is unavailable).
  - **All HTTP methods** ‚Äì GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS with request body support.
  - **Request body support** ‚Äì send JSON, XML, or any data inline or from file with automatic Content-Type detection.
  - **IPv4/IPv6 aware** ‚Äì the resolver and TLS inspector report both the address and its family.
  - **TLS insights** ‚Äì certificate CN, expiry countdown, cipher suite, and protocol version are captured automatically.
  - **Multiple output modes** ‚Äì rich waterfall view, compact single-line summaries, or `-metrics-only` for scripting.
  - **JSON export** ‚Äì persist full step data (including redirect chains) for later processing.
  - **Extensible** ‚Äì clean Protocol interfaces for DNS, TLS, timing, visualization, and export so you can plug in custom behavior.
